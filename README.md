# Synent Task 7 - React Multi-Page Website

A React-based multi-page website for the Synent Technologies internship submission.

## Pages
- Home
- About
- Services
- Contact

## Features
- React navigation with client-side routing behavior
- Responsive layout and menu
- Consistent CSS across pages

## Run
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
